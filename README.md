\# Bluetooth Scanner Tool

\## Description

This project is a simple Bluetooth Low Energy (BLE) scanner developed using Python and the Bleak library. It scans nearby BLE devices and displays their available device name and Bluetooth address.



\## Files

\- `tool\_main.py` - Main Python script for BLE scanning

\- `requirements.txt` - Project dependency list

\- `demo\_notebook.ipynb` - Demo notebook for project explanation and sample execution

\- `.github/workflows/python-ci.yml` - GitHub Actions workflow file
